# Images Directory

Place your images here for the portfolio:

## Required Images

1. **projects-shipped.jpg** - Image for "Projects shipped" widget card
   - Recommended size: 400x300px or similar aspect ratio
   - Will be displayed as a subtle background in the widget card

2. **discord-bots.jpg** - Image for "Discord bots launched" widget card
   - Recommended size: 400x300px or similar aspect ratio
   - Will be displayed as a subtle background in the widget card

3. **fullstack-craftsmanship.jpg** - Image for "Full-stack craftsmanship" highlight card
   - Recommended size: 800x600px or similar aspect ratio
   - Will be displayed as a subtle background in the highlight card

## Image Guidelines

- Use JPG or PNG format
- Optimize images for web (keep file sizes reasonable)
- Images will be displayed with low opacity as background elements
- Text will remain readable over the images

## Alternative Formats

If you prefer different file names or formats, update the image paths in:
- `src/components/Hero.jsx` (for widget cards)
- `src/components/Highlights.jsx` (for highlight card)
