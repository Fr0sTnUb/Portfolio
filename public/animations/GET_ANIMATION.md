# Get Your Rive Animation

## Quick Download Guide

### Option 1: Use a Free Community Animation

1. **Visit**: https://rive.app/community/
2. **Search for**: "abstract background" or "gradient"
3. **Filter**: Free animations only
4. **Download** any suitable animation
5. **Rename** to `hero-bg.riv`
6. **Place** in this folder (`public/animations/`)

### Option 2: Recommended Animations

Search for these specific types on Rive Community:
- **Abstract Backgrounds**: Perfect for hero sections
- **Gradient Waves**: Smooth, subtle animations
- **Particle Effects**: Modern, tech-focused
- **Geometric Patterns**: Clean, professional

### Option 3: Create Your Own

1. Visit https://rive.app/editor
2. Create a simple abstract background
3. Use your brand colors: #6366f1 to #8b5cf6
4. Export as `.riv` file
5. Save as `hero-bg.riv` in this folder

## Once You Have the File

The animation is already enabled in the code! Just:
1. Place `hero-bg.riv` in this folder
2. Refresh your browser
3. The animation will appear automatically

## File Size Tips

- Keep animations under 500KB for best performance
- Use simple animations for backgrounds
- Complex animations work better as foreground elements
