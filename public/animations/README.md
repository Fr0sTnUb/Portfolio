# Rive Animations Folder

Place your `.riv` animation files here.

## Example Structure

```
public/
  animations/
    hero-background.riv
    button-animation.riv
    icon-animation.riv
    loading-spinner.riv
```

## How to Use

1. Download or create Rive animations at [rive.app](https://rive.app)
2. Export as `.riv` files
3. Place them in this folder
4. Reference them in your components as `/animations/filename.riv`
