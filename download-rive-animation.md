# Download Rive Animation

## Quick Setup

1. **Visit Rive Community**: https://rive.app/community/

2. **Recommended Free Animations for Portfolio Hero**:
   - Search for: "background", "abstract", "gradient", "particles"
   - Look for animations that are:
     - Abstract/geometric
     - Purple/blue color scheme
     - Background-style (not too busy)
     - Small file size (< 500KB)

3. **Specific Recommendations**:
   - Search: "abstract background" or "gradient waves"
   - Filter by: Free, Background, Abstract

4. **Download Steps**:
   - Click on an animation you like
   - Click "Download" or "Get File"
   - Save the `.riv` file
   - Rename it to `hero-bg.riv`
   - Place it in `public/animations/hero-bg.riv`

5. **Enable in Code**:
   - The animation is already set up in `Hero.jsx`
   - Just uncomment lines 121-127 once you have the file

## Alternative: Create Your Own

Visit https://rive.app/editor to create a custom animation matching your brand colors (#6366f1 to #8b5cf6).
